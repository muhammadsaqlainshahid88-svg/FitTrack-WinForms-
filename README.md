# FITRACK-WINFORM

 Features

 User Authentication — Register/Login with username & password validation
 Account Lockout — Locks after 3 failed login attempts
 Daily Calorie Goal — Set your burn target and track progress
 Multi-Activity Tracking — Select multiple activities in one session:

 Walking |  Running | Cycling |  Swimming |  Pushups |  Jump Rope


 Calorie Calculator — Activity-specific formulas for accurate burn estimation
 Results View — Today's breakdown with goal achievement status
 Progress History — Full history grouped by date with per-day totals
 JSON Persistence — All user data saved to users.json automatically
 Dark Theme UI — Custom dark color palette with teal accent color
